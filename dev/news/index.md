# Changelog

## mlr3data (development version)

## mlr3data 0.9.0

CRAN release: 2024-11-07

- Readd `ames_housing` data set.

## mlr3data 0.8.0

CRAN release: 2024-11-06

- Move `ames_housing` to mlr3.

## mlr3data 0.7.0

CRAN release: 2023-06-29

- Added dataset `ames_housing`, used in the book about mlr3.
- Added dataset `energy_usage`, used in the book about mlr3.

## mlr3data 0.6.1

CRAN release: 2022-08-15

- Fixed documentation and CRAN notes.
- Added simplified version of the penguins data set as
  `penguins_simple`.
- Added labels to data sets.

## mlr3data 0.6.0

CRAN release: 2022-03-18

- Added simplified version of the penguins data set as
  `penguins_simple`.
- Added labels to data sets.

## mlr3data 0.5.0

CRAN release: 2021-06-29

- Added bike sharing regression task from UCI.
- mlr3 tasks are now loaded lazily.

## mlr3data 0.4.0

CRAN release: 2021-06-08

- Added the Indian liver patient dataset `ilpd`.

## mlr3data 0.3.1

CRAN release: 2021-03-19

- Removed penguins data set; it is now included in mlr3.

## mlr3data 0.3.0

CRAN release: 2020-12-04

- Added `optdigits` data obtained via the `OpenML` package.
- Added `moneyball` data obtained via the `OpenML` package.

## mlr3data 0.2.1

CRAN release: 2020-10-05

- Removed dependency on orphaned package `bibtex`.

## mlr3data 0.2.0

CRAN release: 2020-08-03

- Data sets are now automatically added to the task dictionary if mlr3
  is loaded.
- Added penguins data from the `palmerpenguins` package.

## mlr3data 0.1.0

CRAN release: 2020-02-10

Initial release.
