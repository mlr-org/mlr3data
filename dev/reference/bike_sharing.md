# Bike Sharing Demand

Regression data to predict the total count of bikes rented. Contains 13
features and 17379 observations. Target column is `"count"`.

## Source

<https://archive.ics.uci.edu/ml/datasets/bike+sharing+dataset>

## Pre-processing

- All columns have been renamed.

- `instant`, `"registered"` and `"casual"` column have been removed.

- `"season"` and `"weather"` have been converted to
  [`factor()`](https://rdrr.io/r/base/factor.html).

- `"holiday"` and `"working_day"` have been converted to
  [`logical()`](https://rdrr.io/r/base/logical.html).
